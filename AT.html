Overall System

Our project consists with three subsystems --- a software plug-in, an embedded system, and an analog audio device which is going to be a parametric Equalizer. A traditional analog EQ takes in audio source and send processed/filtered analog signal out (to other audio devices). Usually users would control its filter parameters using knobs/push buttons. However, our device is also going to be controlled digitally. A software plug-in we are designing can take user input and send to an embedded system. The embedded system will then convert them to analog signal to control the parameters the EQ. Audio signal processed by the EQ would also be sent back to the computer via ADC in the embedded system for monitoring.
